== Description ==

Custom fields are display in a table form in post content. Specific custom fields can be selected for display in post content in any theme.

**features**

- Custom fields display in front end without edit existing theme.
- Select which fields become visible.
- Customizes display order.
- Have a table to align things nicely.

== Installation ==


1. Upload the **custom-fields-table-display** folder to the /wp-content/plugins/ directory
